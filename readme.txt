== Description ==

A small plugin to force a user to be redirected to the login page if the user is not logged in and is not an admin.

Useful for sites under a dev environment that only your company and client should have access and not other people that might guess the url. Example: dev.example.com/clientsite/

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy
